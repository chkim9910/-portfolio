PROJECT 02

CHANEL 22 BAG Responsive Website
POINT COLOR

CHANEL 22 BAG 캠페인 페이지의 리뉴얼 사이트입니다.
4명의 CHANEL 앰버서드의 영상과 이미지를 최대로 활용하여 화보집과 같은 분위기를 연출하였습니다.

PROJECT 03

국립발레단 Responsive Website
POINT COLOR
본 사이트는 웹접근성을 준수하였습니다.

본 페이지는 팀 프로젝트로 진행되었습니다.

국내외에 위상을 인정받는 국립발레단의 리뉴얼 사이트입니다.
발레단의 화려한 이미지에 모던한 느낌을 더하기 위해 검정색을 주조색으로 하여 빨강과 보라로 포인트를 주었습니다.
UI 인터페이스는 GSAP ScrollTrigger를 사용하여 자연스러운 스크롤 효과를 주었으며, 아래서 위로 서로 겹쳐져 올라오는 슬라이드 효과를 구현하였습니다. FullCalendar를 사용하여 캘린더의 디자인을 커스텀하고, 캘린더가 동작될 수 있도록 하였습니다. 사용자 환경에 반응할 수 있도록 미디어 쿼리를 사용하여 반응형 웹페이지로 제작되었습니다.
Sub page2를 담당하였습니다.

PROJECT 04

동물자유연대 Responsive Website
POINT COLOR
본 사이트는 웹접근성을 준수하였습니다.

국내의 비영리단체 동물자유연대의 리뉴얼 사이트입니다.
기존의 단순하고 옛스러운 느낌에서 탈피하여 현대적이고 시선을 집중시킬 수 있도록 다양한 인터렉션 요소를 배치하였습니다.
UI 인터페이스는 GSAP ScrollTrigger를 사용하여 스크롤에 따라 역동적이면서도 자연스러운 움직임을 주었습니다. 또 슬라이드에 Swiper, Glide를 사용하여 자연스러운 넘김 효과를 주었으며, Bootstrap을 활용하여 모바일 버전의 Gnb에 자연스러운 드롭다운 효과를 주었습니다.
본 페이지는 그누보드로 제작하였으며, 사용자 환경에 반응할 수 있도록 미디어 쿼리를 사용하여 반응형 웹페이지로 제작되었습니다.
