PROJECT 04

동물자유연대 Responsive Website
POINT COLOR
본 사이트는 웹접근성을 준수하였습니다.

국내의 비영리단체 동물자유연대의 리뉴얼 사이트입니다.
기존의 단순하고 옛스러운 느낌에서 탈피하여 현대적이고 시선을 집중시킬 수 있도록 다양한 인터렉션 요소를 배치하였습니다.
UI 인터페이스는 GSAP ScrollTrigger를 사용하여 스크롤에 따라 역동적이면서도 자연스러운 움직임을 주었습니다. 또 슬라이드에 Swiper, Glide를 사용하여 자연스러운 넘김 효과를 주었으며, Bootstrap을 활용하여 모바일 버전의 Gnb에 자연스러운 드롭다운 효과를 주었습니다.
본 페이지는 그누보드로 제작하였으며, 사용자 환경에 반응할 수 있도록 미디어 쿼리를 사용하여 반응형 웹페이지로 제작되었습니다.
